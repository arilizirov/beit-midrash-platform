# Fundamentals of Data Engineering — notes (original summary)

> The kit's own paraphrase of this book's key principles, to guide judgment.
> This is NOT the book's text. For depth, read the book itself.

Reis & Housley's lifecycle framing of modern data engineering.

- Lifecycle: generation → ingestion → transformation → serving.
- 'Undercurrents' run through all of it: security, data management, DataOps/orchestration, software engineering, architecture.
- Choose boring, proven, scalable tools; optimise for the whole lifecycle, not one stage.
