# Data and Reality — notes (original summary)

> The kit's own paraphrase of this book's key principles, to guide judgment.
> This is NOT the book's text. For depth, read the book itself.

William Kent's meditation on how hard it is to model a messy world.

- A model is a map, never the territory; every schema bakes in assumptions.
- Identity ('is this the same thing?') is genuinely hard and context-dependent.
- Meaning is contextual; categories blur at the edges. Stay humble about your model's fit.
