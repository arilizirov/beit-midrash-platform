# Data Mesh — notes (original summary)

> The kit's own paraphrase of this book's key principles, to guide judgment.
> This is NOT the book's text. For depth, read the book itself.

Zhamak Dehghani's decentralization model for analytical data at scale.

- Domains own their analytical data as PRODUCTS (discoverable, addressable, trustworthy, self-describing).
- A self-serve data platform makes domain ownership feasible.
- Governance is federated and computational — global rules enforced by the platform, not a central team.
*A strong fit only at real organisational scale; overkill for small teams.*
