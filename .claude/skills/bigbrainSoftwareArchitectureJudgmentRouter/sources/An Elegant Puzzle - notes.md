# An Elegant Puzzle — notes (original summary)

> The kit's own paraphrase of this book's key principles, to guide judgment.
> This is NOT the book's text. For depth, read the book itself.

Will Larson on engineering management as systems design.

- Treat the org as a system with constraints; fix the bottleneck, not the symptom.
- Sequence work by leverage; prefer the smallest high-leverage step.
- Distinguish reversible from irreversible decisions; move fast on the former, slow on the latter.
- Migrations are how systems actually change at scale; plan them as first-class work.
